# Uso de IA

Para desarrollar el proyecto se utilizó de diferentes IA's para tener un resultado variado y tratar de evitar los sesgos. Para la planeación se utilizaron: ChatGPT, Gemini y DeepSeek con la finalidad de validar decisiones y aclarar dudas, así como para consular.

Se adjuntan los chats.

ChatGPT: https://chatgpt.com/share/6a5db45c-243c-83e9-aeb9-d9b76569abbe  
Gemini: https://share.gemini.google/T1W8izR5v7VZ  
DeepSeek: https://chat.deepseek.com/share/d61xr0h932l84bcyok  

Así mismo, se utilizaron agentes de IA con el IDE de Antigravity para soporte en la codificación. Los chats se encuentran en la carpeta `IA-chats`. En general se utilizó los agentes de IA para la implementación, mediante la planeación concreta del sistema y mediante el uso de prompts lo más descriptivos posibles para evitar sesgos y obtener un código de mayor calidad. Casi por cada una de las principales funcionalidades se cuenta con un chat de planificación, por ejemplo un chat para el almacenamiento de datos "Implementing Access Control MVP", "BioStar Integration and Reporting" para el tema de la integración con Biostar (mock) y los reportes, "Implementing React Access Control Frontend" para el desarrollo del frontend en React y un chat para la documentación "Documentation for Access Control MVP". 

En este desarrollo la IA jugó un papel importante en la construcción, donde tiene grandes fortalezas ya que la implementación no es tan compleja en el sentido del análisis, sino del conocimiento de la tecnología. El uso de IA agiliza la escritura de código y el uso de buenas prácticas.
